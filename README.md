# 📉 Customer Churn Prediction

A machine learning model to predict whether a customer will leave a service.

---

## 🚀 Features

- 📊 Data preprocessing
- 🤖 ML model training
- 📈 Prediction system
- 📉 Accuracy evaluation

---

## 🧠 Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

---

## ⚙️ Workflow

1. Data cleaning
2. Feature selection
3. Model training (Logistic Regression / Random Forest)
4. Evaluation (Accuracy, Confusion Matrix)

---

## 📊 Results

- Achieved ~85–90% accuracy
- Identified key churn factors
