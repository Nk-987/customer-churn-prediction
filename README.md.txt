# Customer Churn Prediction

## Project Overview
This project predicts whether a customer will churn using machine learning techniques.
The goal is to help businesses identify high-risk customers and improve retention.

## Dataset
- File: churn_modelling.csv
- Rows: 10,000
- Target column: Exited (1 = Churn, 0 = Not Churn)

## Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Encoding
4. Model Training
5. Model Evaluation

## Models Used
- Logistic Regression
- Random Forest
- Gradient Boosting

## Evaluation Metrics
- Accuracy
- Recall
- F1 Score
- ROC-AUC

## How to Run
Open the notebook inside the `notebooks` folder and run all cells.
